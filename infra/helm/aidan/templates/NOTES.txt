AIDAN has been deployed!

1. Get the application URL:
{{- if .Values.ingress.host }}
  http://{{ .Values.ingress.host }}
{{- else }}
  kubectl port-forward svc/client-service 8080:80 -n {{ .Values.namespace }}
  Then open http://localhost:8080
{{- end }}

2. Check pod status:
   kubectl get pods -n {{ .Values.namespace }}

3. View server logs:
   kubectl logs -l app=server -n {{ .Values.namespace }}
